# AI-Naive-Bayes-Diabetes
Tugas ini dibuat untuk memenuhi tugas mata kuliah Kecerdasan Buatan pada materi Lecture 4 tentang Naïve Bayes. Intinya kita disuruh nyoba sendiri gimana cara kerja algoritma Naive Bayes, mulai dari load data, training model, sampai buktiin hasilnya dengan hitung manual pakai rumus dari slide kuliah.
Dataset
Dataset yang dipakai namanya Pima Indians Diabetes Database, isinya data kesehatan 768 pasien wanita. Ada 8 fitur seperti kadar glukosa, tekanan darah, BMI, dan usia. Label akhirnya cuma dua, yaitu diabetes atau tidak diabetes.
